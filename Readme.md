# Stack Data Structure in C

A complete implementation of the Stack Data Structure in C++ language for learning Data Structures & Algorithms (DSA).

This repository contains different stack operations, concepts, and problem-solving programs written during my DSA learning journey.

* LIFO (Last In First Out) Principle
* Real-life Applications of Stack

* Pop Operation

* isEmpty()
* isFull()
* Display Stack Elements

* Fixed Size Stack
